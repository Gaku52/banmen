**デジタル時代の揮発性ワーキングメモリ**

> 4つだけ見える。終わったら消える。

## コンセプト

人間のワーキングメモリは同時に約4つしか保持できない。
既存のタスク管理アプリは「全部記録・全部保持」する。結果、溜まり、開くのが怖くなり、使わなくなる。

盤面は逆のアプローチを取る。

- **NOWスロット**: 1つだけ。今やっていること
- **NEXTスロット**: 最大3つ。次にやること
- **完了したら消える**。履歴なし、統計なし、通知なし
- **開いて3秒で全部見える**

かるた選手が強いのは「覚える力」ではなく「忘れる力」。
前の盤面を捨てて、今の盤面だけに集中する。仕事も同じ。

## 設計原則

1. **引き算の設計** - 機能が少ないことが機能
2. **揮発性** - データを保持しないことが価値
3. **3秒ルール** - 開いて3秒で「今やること」がわかる
4. **ワーキングメモリの科学** - 4つの制約はバグではなく仕様
